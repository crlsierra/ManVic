Global Mangrove Forests Distribution, 2000

DESCRIPTION
This archive contains the Global Mangrove Forests Distribution, 2000 data set, created by Chandra P. Giri et al. (2010). Documentation for data is available at this web site:
http://sedac.ciesin.columbia.edu/data/set/lulc-global-mangrove-forests-distribution-2000

Updates and corrections may still be made. If you should discover any problems or errors, please inform us at: ciesin.info@ciesin.columbia.edu

The archive contains the Global Mangrove Forests Distribution, 2000 grids. The grids are available as compressed geotiffs. To access the grids the contents of the zipfiles must be extracted into a single folder. 

DOWNLOADING DATA
The data can be downloaded via the World Wide Web at http://sedac.ciesin.columbia.edu/data/set/lulc-global-mangrove-forests-distribution-2000. At a resolution of 30m the data is available as 10 deg by 10 deg tiles.

Downloaded files need to be uncompressed using either WinZip (Windows file compression utility) or similar applications before they could be accessed in a Geographic Information System (GIS). Users should expect some increase in the size of downloaded data after decompression. 

DATA GRID VALUES
The downloaded grid values represent the presence or absence of mangrove forests at a pixel. The grid values are 0 and 1. 0 means no mangrove forest present and 1 mangrove forest present.
